# netflix-data-analytics
Exploratory Data Analysis on Netflix Titles dataset using Python, Pandas, Seaborn &amp; Plotly
# 🎬 Netflix Titles — Data Analytics

## 📌 Objective
Analyze Netflix Titles dataset to uncover content trends, geographical
distribution, genre popularity, and audience targeting patterns.

## 📊 Analysis Areas
- Data Cleaning & Feature Engineering
- Content Distribution (Movies vs TV Shows)
- Release Year & Content Addition Trends
- Top 10 Countries by Content
- Content Ratings Analysis (TV-MA, TV-14, etc.)
- Movie Duration & TV Show Seasons Analysis
- Genre Trends Over Time
- Correlation Analysis (Duration vs Rating)

## 🛠️ Tools & Libraries
| Tool | Purpose |
|------|---------|
| Python | Core language |
| Pandas & NumPy | Data manipulation & cleaning |
| Matplotlib & Seaborn | Static visualizations |
| Plotly | Interactive trend charts |

## 💡 Key Insights
- Movies dominate Netflix content vs TV Shows
- Rapid content growth after 2016, peaking around 2019–2020
- USA leads in content production; India is #2
- Most content targets adults (TV-MA) and teens (TV-14)
- Dramas, Comedies & Documentaries are the top genres

## 📁 Dataset
Netflix Titles dataset (netflix_titles.csv)
